# Sentiment Analysis Using Deep Learning

## 📌 Project Description
This project is a deep learning-based sentiment analysis system that classifies text into Positive, Negative, and Neutral sentiments using Natural Language Processing (NLP).

## 🚀 Features
- Text preprocessing and cleaning
- Sentiment classification (Positive / Negative / Neutral)
- Deep learning model for accurate prediction
- Simple and interactive user interface

## 🛠️ Technologies Used
- Python
- Deep Learning
- TensorFlow / Keras
- NumPy, Pandas

## 📊 Output
The system takes user input text and predicts its sentiment in real-time.

## 👩‍💻 Project Type
Academic Machine Learning Project

## 📁 Project Status 
Completed as a learning project in Machine Learning & Deep Learning

## 🎯 Goal
To analyze human emotions from text using AI techniques.
